## Setup and Installation
1. Clone the repository:
    ```sh
    git git@github.com:Saracomethstein/denetTT.git
    cd denetTT
    ```

2. Build project:
    ```sh
    make build
    ``` 

3. Start server:
    ```sh
    make server
    ``` 
   
4. Open new shell and start client:
    ```sh
    make client
    ```
